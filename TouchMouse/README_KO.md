# GazeMou - 로키드 안경용 머리 추적 마우스

로키드 AR 안경(RV101)용 머리 움직임 커서 앱. 내장 자이로스코프로 머리 움직임을 마우스 커서로 변환하여 터치스크린 없는 기기에서 핸즈프리 조작을 지원합니다.

**최신 버전: v1.6.0** | [APK 다운로드](https://github.com/KUPdriveouter/RokidAppMaker/releases/latest)

---

## v1.6.0 주요 변경사항

> v1.5.0은 외부 미공개. 이 릴리즈에 v1.5.0 + v1.6.0 변경사항이 모두 포함되어 있습니다.

### 제스처 개편 (v1.5.0~)
- **셀렉션 모드** — 기존 원형 제스처 제거, 끄덕끄덕(아래-위-아래-위)으로 4방향 메뉴 진입 (좌=스크롤, 우=드웰, 위=커서 끄기, 아래=나가기)
- **도리도리로 커서 켜기** — 커서 꺼진 상태에서 도리도리하면 커서 활성화 (패시브 센서 상시 대기)
- 끄덕끄덕 x2로 스크롤/드웰 모드에서 일반 커서로 복귀

### 어웨이크 모드 (v1.6.0)
- **화면 깨우기** — 화면 꺼진 상태에서 도리도리하면 화면 깨움 + 즐겨찾기 앱 목록 자동 표시
- **오토 커서** 옵션 — 어웨이크 시 커서 + 드웰 모드 자동 활성화
- 화면 꺼진 상태에서는 도리도리(yaw)만 인식, 끄덕끄덕(pitch)은 완전 차단

### 앱 드로어 & 즐겨찾기 (v1.6.0)
- **즐겨찾기 관리** — 설치된 앱 목록에서 탭/클릭으로 즐겨찾기 등록/해제
- **분할 화면 UI** — 상단: 즐겨찾기 (순서 변경 가능) / 하단: 전체 앱 목록
- 런처 내장 앱 (카메라, 번역, AI 챗 등) `[B]` 태그로 구분
- 앱 아이콘 함께 표시
- **순서 조정** — 탭으로 잡기, 스와이프로 이동, 다시 탭으로 놓기
- 커서 클릭 지원 — 드웰 클릭으로 앱 목록 항목 선택 가능

### 오토 포커스 (v1.6.0)
- 커서 멈추면 천천히 화면 중앙으로 이동 (0.6초 대기 후)
- 일반 커서 모드에서만 동작 (드웰/스크롤/셀렉션 모드에서는 비활성)

### 도리도리 인식 튜닝 (v1.6.0)
- 적당한 범위 — 목에 부담 없는 편안한 크기
- 1.5회 패턴 (3회 방향 전환) — 좌-우-좌면 인식
- 최소 250ms 간격 필터 — 걷기/차량 진동에 의한 오작동 방지
- 화면 꺼질 때 제스처 상태 초기화 — 켜질 때 오작동 방지

## 조작법

| 동작 | 제스처 |
|------|--------|
| 커서 토글 | 터치패드 L-L-R-R |
| 커서 켜기 (꺼진 상태) | 도리도리 |
| 셀렉션 모드 | 끄덕끄덕 (아래-위-아래-위) |
| 드웰 클릭 모드 | 셀렉션 > 우 |
| 스크롤 모드 | 셀렉션 > 좌 |
| 커서 끄기 | 셀렉션 > 위 |
| 모드 나가기 | 셀렉션 > 아래 / 끄덕끄덕 x2 |
| 뒤로가기 | 도리도리 (커서 ON + Shake Back 켜진 상태) |
| 화면 깨우기 | 도리도리 (화면 OFF + Awake 켜진 상태) |

## 설정 항목

| 설정 | 설명 |
|------|------|
| Accessibility | 시스템 접근성 설정 열기 |
| Cursor | 커서 켜기/끄기 |
| Speed X / Speed Y | 머리 추적 감도 |
| Hold Time | 드웰 클릭 인식 시간 |
| Cooldown | 다음 드웰까지 대기 시간 |
| Range | 드웰 인식 반경 |
| Shake Back | 도리도리 뒤로가기 (ON/OFF) |
| Awake | 도리도리 화면 깨우기 + 앱 목록 (ON/OFF) |
| Auto Cursor | 어웨이크 시 커서 + 드웰 자동 활성화 (ON/OFF) |
| Auto Focus | 커서 유휴 시 중앙으로 이동 (ON/OFF) |
| Favorites | 즐겨찾기 앱 관리 화면 열기 |

## 설치 방법

1. [Releases](https://github.com/KUPdriveouter/RokidAppMaker/releases)에서 APK 다운로드
2. 로키드 안경에 사이드로드 (ADB 또는 [RokidApkUploader](https://github.com/nicekid1/RokidApkUploader) 사용)
3. 설정 > 접근성 > GazeMou > 활성화
4. 터치패드에서 L-L-R-R로 시작

## 요구사항

- Rokid Glasses RV101 (YodaOS-Sprite, Android 12 / API 32)
- Google Play 서비스 불필요

## 기술 스택

- Kotlin / Android AccessibilityService
- 자이로스코프 센서 기반 머리 추적
- 시스템 오버레이 커서 렌더링
- AndroidX 외 외부 라이브러리 없음
